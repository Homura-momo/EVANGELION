# EVANGELION

## 简介

清华大学“计算机程序设计”课程大作业，一个基于EasyX的射击闯关游戏

游戏以Eva世界观为背景，玩家扮演Eva驾驶员，驾驶Eva与使徒战斗，玩法参考《坦克大战》和《以撒的结合》

## 编译

| 环境   | 配置      |
|------|---------|
| 系统   | Windows |
| 语言   | C++ 14  |
| 开发框架 | EasyX   |
| 编译器  | MSVC    |
| 构建工具 | CMake   |

- 本项目已安装EasyX 2023大暑版，其它版本EasyX图形库可自行安装
    - Visual Studio用户可直接安装：https://easyx.cn/
    - Clion用户可参考：https://zhuanlan.zhihu.com/p/493711304

## 文件结构

| 文件夹     | 说明           |
|---------|--------------|
| src     | 源代码          |
| include | 头文件          |
| images  | 图片素材         |
| sounds  | 音频素材，背景音乐    |
